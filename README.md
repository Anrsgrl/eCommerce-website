## Running Tests

To run tests, run the following command

```bash
  npm run test
```


## Run Locally

Clone the project

```bash
  git clone https://github.com/Anrsgrl/eCommerce-website
```

Go to the project directory

```bash
  cd eCommerce-website
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
